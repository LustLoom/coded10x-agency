# CODED10X Agency Website
**ESTABLISHED MMXXVI**

## Project Overview
CODED10X is a growth-driven creative agency based in **Austin, Texas**. We combine the artistry of cinematic design with the precision of performance engineering to build bespoke digital assets that command authority.

## Technologies Used
* **Frontend Architecture:** HTML5 & CSS3 with custom luxury styling.
* **Motion & Interactions:** GSAP (GreenSock), ScrollTrigger, and SplitType for cinematic text animations.
* **Experience Engineering:** Lenis Smooth Scroll for high-end navigation.
* **Typography:** Montserrat and Playfair Display via Google Fonts.

* `index.html`: The central hub — "We Don't Chase Attention. We Command It."
* `about.html`: Our philosophy as the "Silent Force" in digital performance.
* `services.html`: Breakdown of our Digital Architecture, Conversion Psychology, and Brand Identity.
